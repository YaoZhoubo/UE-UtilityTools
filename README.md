# UtilityTools
记录杂项功能，功能放在插件UtilityTools下，提供一个示例工程`ExampleProject.uproject`。
## 安装
1. 克隆仓库到本地
2. 右键 `ExampleProject.uproject`， 点击`Generate Visual Studio project files` 生成Visual Studio工程文件
3. 打开解决方案`ExampleProject.sln`构建项目  
  
注：UE版本5.5
