# 使用自定义深度模板
UE中可以从FSceneTextures中拿到CustomDepth，它保存了Depth和Stencil，存储了深度和模板，这里要注意，模板的纹理格式是PF_X24_G8，它在低8位存储了模板值，需要用Texture2D<uint2>来接收。
# 使用流程
1. 声明着色器参数

        SHADER_PARAMETER_RDG_TEXTURE_SRV(Texture2D<uint2>, CustomDepthTexture)
1. 获取纹理（C++端）

        void OnResolvedSceneColor_RenderThread(FRDGBuilder& GraphBuilder, const FSceneTextures& SceneTextures)
        {
           PassParameters->CustomDepthTexture = SceneTextures.CustomDepth.Stencil;
        }
2. HLSL着色器处理

        // 声明模板纹理（使用uint2格式）
        Texture2D<uint2> StencilTexture;

        // 采样
        uint2 Stencil = StencilTexture.Load(uint3(SvPosition.xy, 0));
        // 提取模板值（低8位）
        uint stencilValue = CustomDepth.g & 0xFF;
